# bind.sync

与 v-model 类似, 是 $emit('update:[val]', value) 与 props: { [val]: null } 的语法糖

预览效果见左侧 ⬅️

如下代码, 组件 modelBase 上使用了 .sync



 









<template>
    <div class="stages">
        <modelBase :width.sync="vBase"/>
    </div>
</template>

<script>
import KEEP_CLEAR from "./keep-clear";
export default KEEP_CLEAR;
</script>
